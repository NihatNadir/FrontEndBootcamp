Javascript Objects

            const user = {
                name:'John',
            };
            console.dir(user);
            console.log(user);


Object.assign(target, source)

Varolan nesnenin içine başka nesnenin eklenmesini istediğimizde kullanırız. target kısmına eklenmesini istediğimiz object değişkeninin adını yazıyoruz. source kısmına ise ekleyeceğimiz key-value değerlerini yazıyoruz.

            Object.assign(user,{
                property1: 'Value1',
                property2: 'Value2',
                property3: 'Value3',
                property4: 'Value4',
                property5: 'Value5',
            });

Burada user object'ine closure içinde key ve value ekliyoruz.


Object ile ilgili diğer metotlar ve kullanılışları

Object.keys() ile key değerleri yazdırılır.

Object.values() ile value değerleri yazdırılır.

Object.entries() ile hem key hem de value değerleri yazdırılır.

            console.log('Object.keys()',Object.keys(user));
            console.log('Object.values()',Object.values(user));
            console.log('Object.entries()',Object.entries(user));
